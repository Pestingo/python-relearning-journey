# Day 02: Smart Access Control and Profile Manager

## 📝 Overview
Day 02 focused on shifting from linear sequences to complex key-value data mappings, handling unique elements via mathematical sets, and structural conditional logic (`if-elif-else`). This project simulates a dynamic identity and access management system.

---

## 🚀 Key Concepts Covered

### 1. Dictionaries (Structured Data)
* Storing configurations using Key-Value pairs.
* Dynamically updating existing properties and inserting new system metadata at runtime.

### 2. Conditional Logic Control Flow
* Handling branching logic paths efficiently using `if`, `elif`, and fallback `else` structural closures.

### 3. Sets & Set Operations
* Utilizing mathematical array manipulations on unique data collections:
  * **Intersection (`.intersection()`):** Extracting matching security tokens.
  * **Difference (`.difference()`):** Identifying missing clearance keys.
  * **Union (`.union()`):** Merging all distinct tags registered across the system.

---

## 🛠️ Project Structure
* `app.py`: The core application code executing profile management and security tag evaluation logic.
